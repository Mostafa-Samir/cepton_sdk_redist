Capture Replay¶

PCAP capture file replay. Source code can be found in the source folder. Functions are not thread safe, and should only be called from the main thread.

static bool cepton_sdk::capture_replay::is_open()¶
static SensorErrorCode cepton_sdk::capture_replay::open(const std::string &path)¶

Opens capture file.

Must be called before any other replay functions listed below.

static SensorErrorCode cepton_sdk::capture_replay::close()¶
static uint64_t cepton_sdk::capture_replay::get_start_time()¶

Returns capture start timestamp (unix time [microseconds]).

static float cepton_sdk::capture_replay::get_position()¶

Returns capture file position [seconds].

static float cepton_sdk::capture_replay::get_length()¶

Returns capture file length [seconds].

static bool cepton_sdk::capture_replay::is_end()¶

Returns true if at end of capture file.

This is only relevant when using resume_blocking methods.

static SensorErrorCode cepton_sdk::capture_replay::seek(float position)¶

Seek to capture file position [seconds].

Position must be in range [0.0, capture length). Returns error if position is invalid.

static SensorErrorCode cepton_sdk::capture_replay::set_enable_loop(bool value)¶

If enabled, replay will automatically rewind at end.

static bool cepton_sdk::capture_replay::get_enable_loop()¶
static SensorErrorCode cepton_sdk::capture_replay::set_speed(float speed)¶

Replay speed multiplier for asynchronous replay.

static float cepton_sdk::capture_replay::get_speed()¶
static SensorErrorCode cepton_sdk::capture_replay::resume_blocking_once()¶

Replay next packet in current thread without sleeping.

Pauses replay thread if it is running.

static SensorErrorCode cepton_sdk::capture_replay::resume_blocking(float duration)¶

Replay multiple packets synchronously.

No sleep between packets. Resume duration must be non-negative. Pauses replay thread if it is running.

static bool cepton_sdk::capture_replay::is_running()¶

Returns true if replay thread is running.

static SensorErrorCode cepton_sdk::capture_replay::resume()¶

Packets are replayed in realtime. Replay thread sleeps in between packets.

static SensorErrorCode cepton_sdk::capture_replay::pause()¶

Pauses asynchronous replay thread.
